# Weather-Web

## APIs
- **Open Meteo**. <a href="https://open-meteo.com/">Link Page.</a>
- **Countries Database**: <a href="https://open-meteo.com/](https://github.com/dr5hn/countries-states-cities-database">Link Page.</a>
